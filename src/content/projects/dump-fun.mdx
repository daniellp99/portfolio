## **About:**

Shipped token search, trenches chat/tracker, mining and airdrop flows, leaderboards, wallet trackers, and landing animations on a Next.js/React Solana trading frontend; drove performance and UX fixes (Virtuoso, panels, auth redirects).

## **Features:**

* Shipped token search modal, trench chat (timestamps, nicknames, ban flow), and trenches tracker UI on a Solana token trading platform (Next.js 16, React 19).
* Built mining and airdrop flows: mining page/detail/form, timer and rewards grid, animated coins, leaderboard, and daily task verification.
* Implemented leaderboard experience: table, tabs, podium, top unique buyer, animated counting, and monthly token cards (incl. mobile).
* Delivered wallet tracker panel, trades table, and floating panels with drag/resize; optimized table and panel rendering and window-dimension usage.
* Added landing and marketing features: roadmap, GSAP scroll/star animations, countdown, Certik integration, and mobile-responsive ICO layout.
* Fixed layout and scroll behavior (Virtuoso/collapsible heights, overscroll), auth/redirect flows, and ramping/liquidity validation.
